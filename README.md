## Description
This is my personal portfolio website, built as the final project for the Web Development course. It showcases my skills, projects, and contact information. The site is fully responsive and includes interactive features such as dark mode, back‑to‑top button, and form validation.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Tailwind CSS
- Font Awesome & Bootstrap Icons
- Netlify (hosting)
- Git & GitHub

## Features
- Responsive navigation bar with mobile hamburger menu
- Dark / Light mode toggle (persists using localStorage)
- Back‑to‑top button
- Contact form with real‑time validation
- Embedded Google Map
- Project cards with hover effects and technology tags
- Extensive use of icons for better user experience
